# macOS setup notes

1. Cài Python 3.11+, Node.js LTS và FFmpeg, ví dụ qua Homebrew.
2. Chạy `chmod +x scripts/*.sh`.
3. Chạy `./scripts/check_system.sh`.
4. Chạy `./scripts/start_local_app.sh`.

Tính năng Open Folder dùng lệnh hệ thống `open`; Reveal File dùng `open -R`. App mặc định chỉ bind `127.0.0.1`.
